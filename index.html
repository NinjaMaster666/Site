Здравствуй! это гайд о том как же через фишинг написать DOX на обидчика?

У многих бывало когда к примеру оскорбили родных или тебя и ты ничего не можешь этим поделать

Для начала нужно понять как думает этот человек, если он наивный то будет легко.

PC:

1. Установи Kali Linux и зарегистрируйся в нём

2. Установи “Windows subsystem for Linux” с Microsoft Store

3. Активируй “Подсистема Windows для Linux” через Панель Управления

Либо зайдите в PowrShell через администартора и впишите "wsl --install"

4. Перезагрузи компьютер

5. Открой Kali Linux

6. Впиши в терминал 2 команды с этого репозитория GitHub которые указаны в самом начале

7. Зарегистрируемся/Входим на Ngrok и нажимаем “ngrok authotoken” слева сверху рядом с именем пользователя

8. Списываем с Ngrok токен в терминал и курсором тыкаем по “OK”

9. Пишем в терминале “sudo shark”

10. Выбираем любой сервис на которой будем делать фишинг(будем делать на примере 6-Google)

11. a

12. b

13. n

14. y

15. Делаем редирект на “https://www.youtube.com” к примеру “вот чекай видос про тебя)”

16. y

17. 8080

18. Если хотите меняйте ссылку под свою(не указывайте официальные сайты, будет не работать фишинг и выдавать только IP обидчика)

19. Кидаем обидчику ссылку, с текстом из 14 действия

20. Ждём данных

Mobile:

1. Установи Termux

2. Впиши в терминал 2 команды с этого репозитория GitHub которые указаны в самом начале для Termux

4.Списываем с Ngrok токен в терминал и курсором тыкаем по “OK”

5. Выбираем любой сервис на которой будем делать фишинг(будем делать на примере 6-Google)

14. Кидаем обидчику ссылку, с текстом из 9 действия

Готово! если вы успешно забрали учётную запись обидчика то зайдите на неё с VPN, попытайтесь найти максимум данных в его учётной записи начиная с ФИО а позже благополучно снесите ему почту
